01
Zero-Downtime Deploys
Rolling container updates with health checks. Automatic rollback triggers if any health check fails within 60 seconds.
02
End-to-End SSL Encryption
Let's Encrypt certificates, auto-renewed via Certbot. All traffic HTTPS-only with HSTS headers enforced.
03
Automated Security Scans
SAST + dependency audits run on every PR. Docker image scanned for CVEs before any ECR push is allowed.
04
Infrastructure as Code
All server config, environment variables, and deploy scripts version-controlled. Reproducible from scratch in under 5 minutes.

NexaDeploy is my 3rd Year Engineering project — built to demonstrate production-level DevOps practices in a real, deployed environment. From writing the first Dockerfile to configuring SSL on AWS EC2, every component was built from scratch.

The project focuses on what really matters in modern software delivery: automation, security, and reliability. Every push to main triggers a full pipeline — lint, test, build, push, deploy. Zero manual steps. Zero downtime.
